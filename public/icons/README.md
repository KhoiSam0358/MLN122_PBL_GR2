# Icons Guide

## Required PNG Icons

Đặt các file PNG sau vào thư mục này:

### Manual Labor Icons (Bên trái)
- **hammer.png** - Icon búa (Hammer)
  - Kích thước đề xuất: 200x200px hoặc lớn hơn
  - Background: Transparent
  - Màu: Tùy chọn (sẽ có filter overlay)

- **anvil.png** - Icon đe (Anvil) 
  - Kích thước đề xuất: 240x240px hoặc lớn hơn
  - Background: Transparent
  - Màu: Tùy chọn (sẽ có filter overlay)

### Optional Icons (Có thể thêm sau)
- **gear.png** - Icon bánh răng (hiện đang dùng SVG)
- **brain.png** - Icon não bộ (hiện đang dùng SVG)
- **bulb.png** - Icon bóng đèn (hiện đang dùng SVG)
- **tools.png** - Icon công cụ (hiện đang dùng SVG)
- **network.png** - Icon mạng lưới (hiện đang dùng SVG)
